- Vào folder "Assets/Resources" , Chuột phải => Create => SCN => Scriptable Objects => LoadScene
- Sau đó điền các thông tin về:
	- Layer
	- Order in layer
	- Load Anim

Ví dụ: Để chạy được Scene Demo:

1. Kéo prefab trong thư mục LoadScene/Demo vào "Load scene settings" (SO này nằm trong Resources vừa tạo)
2. Sau đó mở Build setting rồi đưa Scene Demo vào
3. Mở Scene Demo lên chạy thử
